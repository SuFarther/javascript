js基本语法

一、基础介绍
  javascript
     网景公司  firebug
  特点： js语法特点也非常灵活
  四种声明方式
     第一种： 在页面的任何地方插入<script>js代码</script>
     第二种:  类似于行内样式 在a标签的href中写入javascript代码
     第三种： 外联引入式 <script src="js.js"></script>
     第四种:  直接给标签添加事件

     三种打印输出方式
       第一种: alert 是一个函数,可以弹窗警告,也经常来调试
       第二种: console.log 打印函数,要配合相关调试工具才能看到
       第三种: document.write() 向浏览器中写入东西

二、基本语法
  1、变量声明
      变量声明: 都要以var开始,但是不以var开始也行，但是我们都要以var开始

      分号结尾: 每一句代码写完之后都要以分号结尾,不以分号结尾也行,但是中间要有回车符，
      但是推荐要以分号结尾,而且以分号结尾有助于代码的压缩
      命名规范：
        1、由数字、字母、下划线、$符号组成
        2、不能以数字开头
        3、不能是系统关键字
        4、严格区分大小写
        5、命名通常以这种标记开始
        int float string array bool object
        iAbc fBca sAbc azBC bABC oDiv

  2、注释格式
    单行注释 //
    多行注释 /**/
    小技巧   (/) /* //*/

   、"undefined"
   字符串
      [注1] js中字符串可以使用单引号,也可以使用双引号

      [注4] 字符串拼接使用 +
   对象、未定义、NaN
     String Number Array Boolean
     未定义: 定义一个变量没有给初始值,那么其就是undefined
     NaN:(not a number)
     当将一个不是数字的字符串转化为数字的时候会产生NaN,NaN用来判断是否相等,判断是不是
     NaN使用isNaN,NaN用来判断是否相等,判断是不是NaN使用isNaN来判断

      6、函数声明
      注意事项
         1、js中函数名字区分大小写
         2、js中函数可以重载,后面的会覆盖前面的
      默认参数
         1、函数中的形参不要加var
         2、有参数的时候调用函数可以传值也可以不传值
      全局变量和局部变量
      1、在函数外面定义的变量都是全局变量,全局变量全局有效,在函数里面直接使用
      2、在函数里面定义的变量叫做局部变量,只能在该函数中使用
      可变长度参数
         在函数内部可以通过 arguments这个对象得到,要想获取参数,下标从0开始
      匿名函数
            var t = function(){
               alert('Closure');
            }
            t();

      封闭空间
      (function (m){
         alert('封闭空间');
         alert(m);
      })(100);
